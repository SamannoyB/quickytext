Welcome Home Page [GET]

API Docs [GET]

curl https://quickytext.samannoyb.hackclub.app/help

That's what you are reading right now! The API documentation.

Make A New Post! [POST]

curl -X POST https://quickytext.samannoyb.hackclub.app/createpost -H "Content-Type: application/json" -d '{"name":"John", "email":"john@example.com", "title":"Hello World", "content":"This is a test post."}'

Make a new post! Required parameters are name, email, title and content.

Access All Posts [GET]

curl https://quickytext.samannoyb.hackclub.app/posts

All posts of QuickyText in one place!

Filter Posts By Email [GET]

curl https://quickytext.samannoyb.hackclub.app/posts/email/[email]

Put a specific email address in place of [email]! Filter posts for that email.

Search for all posts [GET]

curl https://quickytext.samannoyb.hackclub.app/posts/search

Search for all posts in the QuickyText microposting app!
